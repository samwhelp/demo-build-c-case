# 安裝 build-essential


## 環境

* Xubuntu 20.04


## 安裝指令

執行下面指令， 安裝「Package: [build-essential](https://packages.ubuntu.com/focal/build-essential)」。

``` sh
$ sudo apt-get install build-essential
```
